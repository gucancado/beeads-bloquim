
criar funcionalidade de download de anexos em tarefas

Objetivo:
Permitir que o usuário faça download dos arquivos anexados dentro da tela de edição de tarefa, de forma simples e intuitiva.

Requisitos funcionais:

1. Ação de download

* Cada arquivo anexado deve ter a funcionalidade de download disponível.
* O download deve baixar o arquivo original, preservando nome e formato.

2. Formas de realizar download

* Duplo clique no anexo

  * Ao dar duplo clique no elemento do arquivo, o download deve ser iniciado automaticamente.
  * Ao passar o cursor sobre o anexo, deve haver um feedback visual indicando que o item é clicável (ex: mudança de cursor ou leve destaque).

* Botão de download

  * Cada anexo deve ter um botão de download visível.
  * O botão deve ficar posicionado à esquerda do ícone de excluir anexo.
  * O botão deve ter um ícone padrão de download.
  * Ao clicar no botão, o download deve iniciar imediatamente.

3. Comportamento da interface

* O download não deve redirecionar o usuário para outra página.
* O download deve ocorrer diretamente no navegador.
* A interação deve ser rápida, sem etapas intermediárias desnecessárias.
* O usuário deve conseguir identificar facilmente a ação de download.

4. Estrutura técnica

* O sistema deve recuperar o arquivo armazenado no backend e disponibilizá-lo para download.
* Garantir que o endpoint de download retorne o arquivo com os headers corretos para:

  * forçar download (Content-Disposition)
  * manter nome original do arquivo
* Garantir suporte a diferentes tipos de arquivos (imagens, PDFs, etc).

5. Regras e cuidados

* O download deve funcionar para todos os tipos de arquivos suportados pelo sistema.
* Garantir que arquivos grandes não quebrem a experiência (lidar com tempo de resposta).
* Evitar abrir arquivos diretamente em nova aba, priorizar download.

6. Critérios de aceite

* Consigo baixar um anexo com duplo clique.
* Consigo baixar um anexo pelo botão de download.
* O arquivo é baixado com o nome correto.
* O download funciona para diferentes tipos de arquivo.
* A interface deixa claro que o anexo é interativo.
* O botão de download está corretamente posicionado ao lado do botão de excluir.
