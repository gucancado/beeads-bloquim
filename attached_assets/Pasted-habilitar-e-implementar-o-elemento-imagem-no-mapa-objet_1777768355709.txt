habilitar e implementar o elemento "imagem" no mapa

objetivo:
permitir inserir, visualizar e manipular imagens diretamente no mapa, integrando com o sistema de anexos

1) habilitação do botão

- o botão "imagem" já existe no menu de ferramentas
- remover estado desabilitado
- garantir que o botão esteja clicável e funcional

---

2) criação do elemento imagem

- ao clicar no botão "imagem":
  - abrir seletor de arquivos do sistema
  - permitir selecionar uma imagem

- após seleção:
  - inserir a imagem no mapa como um novo elemento

---

3) colar imagem via clipboard (ctrl+v)

- ao pressionar ctrl+v com uma imagem na área de transferência:
  - inserir automaticamente a imagem no mapa

- comportamento:
  - a imagem deve ser criada na posição atual do cursor ou no centro da viewport

---

4) vínculo com anexos

- toda imagem criada dentro de um mapa deve:
  - ser salva como um anexo vinculado ao plano

- esse anexo deve também:
  - estar associado ao elemento imagem no mapa

- relação:
  - elemento do mapa → referencia o anexo
  - anexo → mantém dados do arquivo (url, nome, etc.)

---

5) comportamento do elemento imagem

- permitir:
  - mover a imagem no mapa
  - redimensionar a imagem

- regra de redimensionamento:
  - manter sempre a proporção original (aspect ratio fixo)
  - não permitir distorção

---

6) camadas (z-index)

- o elemento imagem deve respeitar a seguinte ordem de camadas:

  - acima de:
    - retângulos
    - elipses

  - abaixo de:
    - linhas

- garantir consistência com o sistema de renderização atual

---

7) interação adicional

- ao selecionar a imagem:

  - permitir copiar a imagem:
    - ao usar comando de copiar (ctrl+c)
    - a imagem deve ser enviada para a área de transferência

  - exibir botão de download:
    - posicionar no canto superior direito da imagem
    - visível apenas quando a imagem estiver selecionada

  - ao clicar no botão de download:
    - baixar o arquivo original da imagem

---

8) estrutura de dados

- o elemento imagem deve armazenar:
  - referência ao anexo (attachmentId ou equivalente)
  - posição (x, y)
  - dimensões (width, height)
  - proporção original

---

9) regras adicionais

- não permitir criação de imagem sem vínculo com anexo
- garantir performance com múltiplas imagens no mapa
- manter consistência visual com os demais elementos do mapa
- não alterar comportamento de outros tipos de elemento

---

resultado esperado:

- botão "imagem" funcional no menu de ferramentas
- usuário consegue adicionar imagens ao mapa via botão ou ctrl+v
- imagens ficam vinculadas ao plano como anexos
- imagens podem ser manipuladas visualmente sem distorção
- imagens respeitam a ordem correta de camadas no mapa
- usuário pode copiar ou baixar imagens facilmente
